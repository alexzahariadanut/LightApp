LightApp este un proiect educațional orientat pe DevOps practices, având ca scop construirea unei aplicații production-ready, folosind containerizare, CI/CD, Kubernetes și Infrastructure as Code.

Proiectul va fi extins gradual pentru a simula un mediu real de producție.

Aplicatia va fi dezvoltata in python.
